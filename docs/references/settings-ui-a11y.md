# Settings UI — references (official docs)

- **Switch pattern**: [MDN — `switch` role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/switch_role) — use `role="switch"` and `aria-checked`.
- **State**: [React — `useState`](https://react.dev/reference/react/useState) — toggles and export format selection.

Implementation: one outer `button` per row (no nested buttons), track + thumb as decorative `span` inside.
