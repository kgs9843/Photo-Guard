# Pull request 본문 템플릿 (SoT)

에이전트·사람 모두 PR 설명은 **이 파일의 섹션 제목·순서·체크리스트**를 유지한 채 내용을 채웁니다.

- **권장:** 내용을 채운 복사본을 루트 `pr-body.local.md` 등으로 저장한 뒤 `bash scripts/pr-open.sh <브랜치> pr-body.local.md`로 PR을 엽니다(`docs/workflows/README.md` 참고).
- **생략 시:** `bash scripts/pr-open.sh`만 실행하면 이 파일 **원문**이 그대로 PR 본문으로 올라갑니다(빈 칸·체크 미표시).

---

## 📝 변경 사항

이 PR에서 변경된 내용을 설명해주세요.

## 🔗 관련 이슈

관련된 이슈 번호를 작성해주세요.

- Closes #(이슈 번호)

## 🎯 변경 유형

해당하는 항목에 체크해주세요:

- [ ] 🐛 버그 수정
- [ ] ✨ 새로운 기능 추가
- [ ] ♻️ 리팩토링
- [ ] 📝 문서 수정
- [ ] 🎨 스타일 변경
- [ ] ⚡️ 성능 개선
- [ ] 🧪 테스트 추가/수정
- [ ] 🔧 빌드/설정 변경

## 🎯 플랫폼

변경 사항이 적용된 플랫폼을 선택해주세요:

- [ ] 📱 모바일
- [ ] 💻 웹
- [ ] 🔄 공통 (모바일 + 웹)

## 📸 스크린샷

변경 사항에 대한 스크린샷을 추가해주세요.

## 📝 추가 정보

리뷰어가 알아야 할 추가 정보가 있다면 작성해주세요.
